1. git status - проверяет статус разработки файлов;
2. git add .  - подготавливает файлы для записи, при помощи "." можно добавить все файлы сразу;
3. git commit -m"coment"  - запись изменений;
4. git log / git log --oneline   - информацио о коммитах / красткая инфо о коммитах;
5. git push [rep_link] [branch_name]  - для загрузки данных на удаленный репозиторий;

6. git push origin master - стандартная запись данных по ссылке origin на ветку master;



7. git reset [file] - позволяет убрать файл из stage(промежуточной области);
8. git diff - позволяет просмотреть те строки, которые мы изменили или добавили;
9. git reset --hard  - отменяет изменения и полностью очищает git status


10.  .gitignore  - файл, в котором указываются папки либо файлы, которые не будут отправляться на наш github;
